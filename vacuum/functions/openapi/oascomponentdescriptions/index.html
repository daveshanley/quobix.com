oasComponentDescriptions

oasComponentDescriptions will check each description found in Components

Example ruleset configuration

description-duplication:
  description: "Check component description"
  type: "validation"
  recommended: true
  severity: "info"
  given: "$"
  then:
    function: "oasComponentDescriptions"

The function is used by the component-description Rule
